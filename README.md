boilerplate isomorphic typescript

# Requirements/Recommended

 - [node] `v6+`
 - [atom]
    - Packages
      - [atom-typescript]
      - [atom-lint]
      - [linter-tslint]
 - [docker]

# Quick start

```bash
# first time
npm install

# every time
npm run server:dev

# for testing - browser debugging at http://localhost:9876/debug.html
npm run test:watch

```

# Features

 - [typescript] - javascript type support
