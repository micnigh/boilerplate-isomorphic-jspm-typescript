boilerplate isomorphic typescript

# Requirements/Recommended

 - [node] `v4+`
 - [atom]
    - Packages
      - [atom-typescript]
      - [atom-lint]
      - [linter-tslint]

# Quick start

```bash
# first time
npm install -g gulp tsd
npm install

# every time
gulp --gulpfile gulpfile.transpile.js watch
gulp watch
```

# Features

 - [typescript] - javascript type support
 - [gulp] - automated build system
